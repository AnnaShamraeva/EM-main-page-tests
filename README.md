Автоматизация тестирования с использованием Python, POM и Allure
Задача: покрыть тестами разделы главное страницы EM: UI тест(-ы):
тесты на главную страницу сайта,
проверяющие переход по всем блокам по клику (О нас, Контакты и пр.).
Проверять нужно соответствующие локаторы и url’ы.

Требования:
● Для UI теста можно использовать Playwright.
● Python версии 3.10
● Все зависимости должны быть описаны в requirements.txt.
● Проект должен быть легко запускаемым на любом компьютере с подробной
инструкцией в README.md.
● Использовать лучшие практики разработки и автоматизации тестирования.
Ожидаемый результат: Репозиторий с проектом, включающий все необходимые
инструкции для установки и запуска.

## <h> Для запуска: </h>

### <h>1. Установите зависимости:</h>

> pip install -r requirements.txt</h>

### <h>2. Запустить все тесты и записать отчет:</h>

> pytest --alluredir=./allure-results

### <h>3. Посмотреть отчет по прогону html</h>

> allure serve ./allure-results
